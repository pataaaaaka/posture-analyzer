# 🚀 クイックスタートガイド

## 5分で始める姿勢解析アプリ

### ステップ1: ファイルの準備 📁

1. `posture-analyzer` フォルダ全体をダウンロード
2. 以下のファイルが含まれていることを確認:
   - `index.html`
   - `style.css`
   - `main.js`
   - `README.md`
   - `IMPROVEMENTS.md`（改善提案）

### ステップ2: アプリを起動 🌐

#### 方法A: 直接開く（簡単）
1. `index.html` をダブルクリック
2. ブラウザが自動的に開きます
3. ⚠️ カメラ機能は使えません（ファイルアップロードのみ）

#### 方法B: ローカルサーバー（推奨）
カメラ機能を使うには HTTPS が必要です。

**Python がある場合:**
```bash
cd posture-analyzer
python -m http.server 8000
```
ブラウザで `http://localhost:8000` にアクセス

**Node.js がある場合:**
```bash
cd posture-analyzer
npx http-server
```

#### 方法C: GitHub Pages（本番環境）
1. GitHubアカウントを作成（無料）
2. 新しいリポジトリを作成
3. ファイルをアップロード
4. Settings > Pages でブランチを選択
5. 公開URLにアクセス

### ステップ3: マーカーの準備 🔴

以下のいずれかを用意:
- **丸いシール**（赤色、直径1cm以上）
- **赤いビニールテープ**（小さく丸めて）
- **赤いマスキングテープ**
- **赤いマーカーペン**（肌に直接描く）

### ステップ4: マーカーを貼る 📍

#### 正面撮影の場合
体の前面、以下の8箇所に赤マーカーを貼る:

```
    👂        👂
     |         |
    🔴--------🔴  ← 左右の肩峰（肩の外側）
     |         |
     |         |
    🔴--------🔴  ← 左右の前上腸骨棘（骨盤の出っ張り）
     |         |
     |         |
    🔴--------🔴  ← 左右の膝の中央
     |         |
     |         |
    🔴--------🔴  ← 左右のくるぶし
```

**コツ:**
- 服は脱いで、肌に直接貼る
- 左右対称に貼る
- はっきり見える位置に

#### 側面撮影の場合
体の側面、以下の5箇所に赤マーカーを貼る:

```
👂 🔴  ← 耳の穴
 |
🔴  ← 肩峰
 |
🔴  ← 大転子（太ももの付け根外側）
 |
🔴  ← 膝の中央
 |
🔴  ← 外くるぶし
```

### ステップ5: 撮影 📸

1. **明るい場所**で撮影（窓際がベスト）
2. **白い壁**の前に立つ
3. **全身**が入るように距離を調整
4. カメラを**水平**に保つ
5. 「写真を撮影/選択」ボタンをタップ

**良い撮影例:**
- ✅ 全身が入っている
- ✅ マーカーがはっきり見える
- ✅ 背景がシンプル
- ✅ 照明が明るい

**NG例:**
- ❌ 体の一部が切れている
- ❌ マーカーが見えにくい
- ❌ 背景がごちゃごちゃ
- ❌ 逆光で暗い

### ステップ6: 解析 🔍

1. 撮影種類を選択（正面/側面/足）
2. 「自動解析」ボタンをタップ
3. 数秒待つ
4. 結果が表示されます！

### ステップ7: 補正（必要に応じて）✏️

マーカー位置がずれている場合:
1. 「手動補正」ボタンをタップ
2. マーカーを**ドラッグ**して移動
3. 位置を調整
4. 「補正完了」をタップ

### ステップ8: 結果の確認 📊

以下の項目が評価されます:

#### 正面撮影
- **肩の傾斜**: 左右の高さの差
- **骨盤の傾斜**: 骨盤の左右差
- **脚のアライメント**: O脚/X脚の判定

#### 側面撮影
- **頭部前方偏位**: 頭が前に出ているか
- **背中の丸み**: 猫背の程度
- **骨盤の姿勢**: 前傾/後傾

#### 評価基準
- 🟢 **良好**: このまま維持！
- 🟡 **要注意**: 少し気をつけましょう
- 🔴 **要改善**: 改善の余地あり

### ステップ9: 保存 💾

1. 「結果画像を保存」ボタンをタップ
2. 画像がダウンロードされます
3. スマホのギャラリーで確認

### ステップ10: 次の撮影 ➡️

「次の撮影へ」ボタンで、正面→側面→足の順に進みます。

## ⚡ よくある質問 (FAQ)

### Q1: マーカーが検出されません
**A:** 以下を確認してください:
- 本当に**赤い**マーカーですか？（ピンクや蛍光色はNG）
- 照明は明るいですか？
- マーカーは大きい（直径1cm以上）ですか？
- 「手動補正」モードで手動追加してみてください

### Q2: カメラが起動しません
**A:** 
- ブラウザのカメラ権限を確認
- HTTPSでアクセスしているか確認（http://ではNG）
- 別のブラウザで試してみる
- ファイルアップロード機能を使う

### Q3: 処理が遅いです
**A:**
- 画像サイズを小さくする（1200x1600px程度が最適）
- 他のアプリを閉じる
- Wi-Fi接続を確認（初回のみ）
- 古い端末の場合、処理に時間がかかります

### Q4: 結果は正確ですか？
**A:**
- このアプリは**教育・フィットネス目的**です
- 医療診断ツールではありません
- 参考値として活用してください
- 異常がある場合は専門医に相談を

### Q5: データは保存されますか？
**A:**
- すべての処理はブラウザ内で完結
- 画像は外部に送信されません
- プライバシー保護を重視しています
- 保存した画像はあなたのデバイス内のみ

### Q6: 無料で使えますか？
**A:**
- **完全無料**です！
- 制限なし、広告なし
- オープンソース（改変OK）
- 商用利用も可能

### Q7: 服を着たままでも大丈夫？
**A:**
- 服の上からマーカーを貼ってもOK
- ただし精度は低下します
- 可能なら肌に直接貼るのがベスト
- タンクトップ＋短パンがおすすめ

### Q8: 一人で撮影できますか？
**A:**
- スマホスタンド・三脚があると便利
- セルフタイマー機能を使う
- 鏡を見ながら撮影
- 家族や友人に頼むのが確実

### Q9: 改善のアドバイスはありますか？
**A:**
- 現バージョンは評価のみ
- 今後、エクササイズ提案機能を追加予定
- 異常がある場合は理学療法士に相談を

### Q10: エラーが出ます
**A:**
- ブラウザのキャッシュをクリア
- ページを再読み込み（F5）
- 別のブラウザで試す
- GitHubで報告していただけると助かります

## 🎯 撮影のコツ

### 環境設定
- **明るさ**: 窓際の自然光がベスト
- **背景**: 白い壁が理想
- **距離**: 2〜3メートル離れる
- **高さ**: カメラを胸の高さに

### マーカー配置
- **サイズ**: 直径1〜2cm
- **色**: 真っ赤（RGB: 255, 0, 0に近い）
- **形**: 丸いシールが最適
- **位置**: 骨の目印に正確に

### 姿勢
- **自然**: 普段の姿勢で
- **リラックス**: 力を抜いて
- **正面**: カメラを真正面に見る
- **側面**: 横を向く（耳が見える角度）

### 注意点
- ⚠️ 撮影直前にストレッチしない
- ⚠️ 靴は脱ぐ
- ⚠️ 厚着しない
- ⚠️ 反り返らない
